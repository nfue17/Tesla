# Tesla vs GameStop: Stock Price and Revenue Analysis

This project analyzes the relationship between stock prices and company revenue for two well-known companies: **Tesla** and **GameStop**.

The goal is to demonstrate how data can be collected from multiple sources, cleaned, and visualized to explore business-related questions.

---

## Project Overview

In this project, stock price data and revenue data were collected and analyzed using Python.  
Stock prices were retrieved via an API, while revenue data was extracted through web scraping.

The final result is an interactive visualization showing the relationship between stock prices and company revenue over time.

![Tesla vs GameStop Visualization](tesla_gme_plot.png)
---

## Data Sources

- **Stock Price Data:** Retrieved using the `yfinance` Python library.
- **Revenue Data:** Extracted from HTML tables using `BeautifulSoup` and `pandas`.

Sources used in this project:

- Tesla stock data via Yahoo Finance API
- GameStop stock data via Yahoo Finance API
- Revenue data from publicly available financial data tables

## Project Workflow

The project follows a typical data analysis workflow:

1. **Import libraries**
2. **Retrieve stock price data** using the yfinance API
3. **Scrape revenue data** from HTML tables using BeautifulSoup
4. **Clean and prepare the data**
5. **Convert data types for analysis**
6. **Create visualizations** comparing stock prices and revenue

---

## Example Visualization

The project generates interactive plots showing:

- Historical **stock price development**
- Historical **company revenue**

This allows an exploration of how company performance and stock prices evolved over time.
